# TriStrike – Advanced Tic Tac Toe Game

TriStrike is a front-end web-based Tic Tac Toe game featuring an intelligent computer opponent, responsive design, smooth UI interactions, and a modern white-orange themed interface. It is built using HTML, CSS, and JavaScript.

## Live Demo
https://parthp137.github.io/TriStrike/

## Features

### Game System
-Player vs Player mode
-Player vs Computer mode (AI using minimax logic)
-Automatic winner detection
-Winner popup message display
-Draw detection with score tracking
-Restart and reset score controls

###UI Elements
-Interactive 3x3 game board
-Real-time turn indicator
-Scoreboard for X, O, and Draw
-Game control panel
-Responsive layout for all screen sizes

###Functionality
-Smart AI move calculation
-Keyboard support for quick moves (1-9 keys)
-Dynamic turn switching
-Winner popup notifications:
"X Won"
"O Won"
"Computer Won"

##Tech Stack
Frontend: HTML, CSS, JavaScript

Tools: VS Code, Git, GitHub Pages

##Design Overview
The website uses a clean white-orange modern theme and includes the following primary components:

-Hero section with branding
-Interactive Tic Tac Toe board
-Control panel with mode selection
-Scoreboard and game status display
-Responsive navigation bar

All gameplay logic and animations are handled using JavaScript.

##Limitations
-No multiplayer over network
-No account or player history system
-No game replay feature
-Single-page structure (no routing)
